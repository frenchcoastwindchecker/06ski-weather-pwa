ℹ️ Notes

  • Recent snowfall = precipitation in the last 3 hours, not the total accumulated snow base.
  • For actual snow height and piste conditions, consult the official bulletins above.
  • Forecast Timeline merges the freezing-level forecast (Open-Meteo) and precipitation events (OWM) into a single chronological table. Rows at the same hour are merged.
  • Freezing level (FL) is sourced from Open-Meteo (free, no API key, ~1–2 km resolution). If unavailable, it is estimated via lapse rate (6.5 °C/1 000 m) from the bottom-station temperature.
