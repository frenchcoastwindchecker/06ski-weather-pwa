ℹ️ Important Notes

  • Recent snowfall shows precipitation in the last 3 hours, NOT the total accumulated snow base.
  • For actual snow height and piste conditions, check the official bulletins above.
  • Forecast Timeline combines the freezing-level forecast (Open-Meteo) and precipitation events (OWM) into a single chronological table. Rows with the same hour are merged.
  • Freezing level (FL) is provided by Open-Meteo (free, no API key, ~1–2 km resolution). If unavailable, it is estimated via lapse rate (6.5°C/1000m) from the bottom-station temperature.
